Cyrano Copilot Icon

Simplified C-Shape Design - HAL Icon with Opening at 3 O'Clock

Design: Based on the original HAL 9000 tribute icon with a gap/break at the 3 o'clock position in both the circle and rectangle. This creates a "C" shape where negative space connects from inside (between the red dot and circle) to the outside, echoing the Cognisint logo.

Design Notes

Key features:

  • Same tall rounded rectangle as original HAL icon
  • Gap at 3 o'clock position in both circle and rectangle
  • Small red dot (radius 1.6) - same size as original
  • Stroke-based design maintains high visual clarity
  • Negative space creates the "C" shape
  • Simple, clean, recognizable at all sizes

Usage: Copilot/assistant entry points in LexFiat and Arkiver
