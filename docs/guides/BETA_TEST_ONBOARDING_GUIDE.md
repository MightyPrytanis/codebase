# Beta Test Onboarding Guide

**Date:** 2025-12-29  
**Audience:** Beta Testers  
**Purpose:** Step-by-step guide for beta testers to register and get started

---

## Quick Start (Single Email Link)

### Step 1: Receive Invitation

You'll receive an email invitation with:
- Subject: "You're Invited to Beta Test Cyrano"
- Unique invitation link
- Clear instructions

### Step 2: Click Invitation Link

Click the invitation link in your email. This will take you to:
`https://cognisint.com/beta?token={your-invitation-token}`

### Step 3: Create Account

1. Your email is pre-filled from the invitation
2. Create a secure password
3. Accept terms and conditions
4. Click "Create Account"

**Time:** < 2 minutes

### Step 4: Install Applications

1. Click "Install" button on dashboard
2. System detects your platform automatically
3. Download installer
4. Run installer (guided steps)
5. Verify installation

**Time:** < 5 minutes

### Step 5: Launch and Explore

1. Launch Cyrano/LexFiat/Arkiver
2. Complete first-time setup wizard
3. Start exploring features
4. Submit feedback as you test

**Total Setup Time:** < 10 minutes

## Detailed Steps

### Registration

**What You Need:**
- Invitation email
- Valid email address
- Ability to create password

**Process:**
1. Click invitation link
2. Verify your email (pre-filled)
3. Create password (minimum 8 characters)
4. Accept terms
5. Click "Create Account"

**Troubleshooting:**
- **"Invalid invitation"** - Contact support or request new invitation
- **"Email already registered"** - You may already have an account. Try logging in instead.

### Installation

**System Requirements:**
- macOS 12+ / Windows 10+ / Linux (Ubuntu 20.04+)
- 4GB RAM minimum
- 2GB free disk space
- Internet connection

**Installation Steps:**

1. **Download Installer**
   - Click "Install" on dashboard
   - System detects your platform
   - Download starts automatically

2. **Run Installer**
   - macOS: Open downloaded .dmg, drag to Applications
   - Windows: Run .exe installer, follow prompts
   - Linux: Run install script, follow prompts

3. **Verify Installation**
   - Installer runs verification automatically
   - Green checkmark = Success
   - If errors, see Troubleshooting section

4. **Launch Application**
   - Click "Launch" button
   - Or find in Applications/Start Menu
   - First-time setup wizard starts

**Troubleshooting:**
- **Installation fails** - Use Cyrano Pathfinder support (click "Get Help" button)
- **Verification fails** - Check system requirements, try again
- **Can't find application** - Check Applications folder (macOS) or Start Menu (Windows)

### First-Time Setup

1. **Welcome Screen**
   - Read welcome message
   - Click "Get Started"

2. **Account Connection**
   - Enter your beta tester email
   - Enter password
   - Click "Connect"

3. **Preferences**
   - Choose UI mode (Full Stack or Essentials)
   - Select theme
   - Configure notifications

4. **Complete Setup**
   - Review summary
   - Click "Complete Setup"
   - You're ready to test!

### Common Questions

**Q: How long is the beta test?**
A: Beta test runs for 90 days. You'll receive updates and can continue testing.

**Q: What should I test?**
A: Test all features, especially:
- Registration and onboarding
- Core workflows
- UI/UX experience
- Performance
- Any edge cases you find

**Q: How do I submit feedback?**
A: Click "Feedback" in dashboard, or use Cyrano Pathfinder to submit feedback.

**Q: What if I find a bug?**
A: Submit via Feedback form with:
- Description of bug
- Steps to reproduce
- Expected vs actual behavior
- Screenshots if possible

**Q: Can I invite others?**
A: Beta testing is invite-only. Contact support if you'd like to recommend someone.
