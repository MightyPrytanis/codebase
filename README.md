# Codebase - Project Cyrano

This repository contains projects developed under the Cyrano architecture framework.

## About Project Cyrano

Project Cyrano follows a modular architecture philosophy that emphasizes clear communication, separation of concerns, and elegant design patterns. All code in this repository must comply with the architectural principles defined in our architecture document.
